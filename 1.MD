## 1ª) Aplicação de Conversão de Tipos Numéricos

**Objetivo:** Criar uma aplicação que receba um valor através de `input()` e converta para número inteiro e ponto flutuante.

**Funções a utilizar:**
- `input()` - Para capturar entrada do usuário
- `int()` - Para converter para número inteiro
- `float()` - Para converter para número de ponto flutuante
- `print()` - Para exibir os resultados

## 2ª) Aplicação que verifica se o nome correto foi digitado

**Objetivo:** Criar uma aplicação que receba um valor através de `input()` e valide através do `if`, `else` e `elif` se está digitado corretamente.

**Funções a utilizar:**
- `input()` - Para capturar entrada do usuário
- `if` - Para fazer uma validação
- `elif` - Para fazer uma validação em caso da anterior ter dado falso
- `else` - Para fazer executar algo em caso de todos os `if` ou `elif` terem dado falso
- `print()` - Para exibir os resultados

## 3ª) Jogo "Complete a palavra"

**Objetivo:** Criar uma aplicação que mostre uma palavra faltando uma sílaba, Ex.: "BA__TA"; e solicite que o usuário complete a palavra com a sílaba certa. Se estiver correto, deve mostrar uma mensagem positiva e a palavra completa; se não, uma mensagem negativa.

**Funções a utilizar:**
- `input()` - Para capturar entrada do usuário
- `if` - Para fazer uma validação
- `elif` - Para fazer uma validação em caso da anterior ter dado falso
- `else` - Para fazer executar algo em caso de todos os `if` ou `elif` terem dado falso
- `print()` - Para exibir os resultados

## 4ª) Cadastro de Usuário em uma lista e dicionário

**Objetivo:** Criar uma aplicação que receba dados do usuário (nome, email, idade) e armazene em um dicionário, depois exiba os dados cadastrados.

**Funções a utilizar:**
- `while True` - Para loop contínuo
- `input()` - Para capturar entrada do usuário
- `dict` - Para armazenar os dados do usuário
- `list` - Para armazenar múltiplos usuários
- `print()` - Para exibir os resultados
